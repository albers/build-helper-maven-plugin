build-helper:uptodate-properties

Full name:

org.codehaus.mojo:build-helper-maven-plugin:3.1.0:uptodate-properties

Description:

Sets multiple properties according to whether multiple sets of source and target resources are respectively up to date.

Attributes:

  • Requires a Maven project to be executed.
  • Since version: 1.12.
  • Binds by default to the lifecycle phase: validate.

Optional Parameters

Name Type Since Description
skip boolean 1.12 Disables the plug-in execution.
Default value is: false.
User property is: buildhelper.uptodateproperties.skip.
upToDatePropertySettings List 1.12 List of UpToDatePropertySettings to apply.

Parameter Details

skip:

Disables the plug-in execution.
  • Type: boolean
  • Since: 1.12
  • Required: No
  • User Property: buildhelper.uptodateproperties.skip
  • Default: false

upToDatePropertySettings:

List of UpToDatePropertySettings to apply.
  • Type: java.util.List
  • Since: 1.12
  • Required: No
